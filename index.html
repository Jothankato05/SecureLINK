Use the mobile app, web platform, or USSD code to send an SOS alert instantly.
Is my location shared securely?
Yes, all location data is encrypted and only shared with verified responders.
Can I customize my trusted contacts?
Yes, you can manage your trusted contact list in your profile settings.
